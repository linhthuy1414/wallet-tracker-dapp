# Wallet Tracker DApp

Một dự án blockchain nhỏ mô phỏng bảng điều khiển theo dõi ví.

## Chức năng hiện có

- Nhập địa chỉ ví để tải dữ liệu mô phỏng
- Hiển thị số dư ước tính
- Thống kê tổng số giao dịch
- Phân loại giao dịch nhận, gửi và gọi hợp đồng
- Lọc danh sách giao dịch theo loại
- Giao diện dạng dashboard để trưng bày trên GitHub

## Công nghệ sử dụng

- HTML
- CSS
- JavaScript

## Cấu trúc thư mục

```text
wallet-tracker-dapp/
├── index.html
├── style.css
├── script.js
└── README.md
```

## Cách chạy

1. Tải mã nguồn về máy
2. Mở file `index.html` bằng trình duyệt
3. Nhập một địa chỉ ví bất kỳ để xem dữ liệu mô phỏng

## Hướng phát triển tiếp

- Kết nối API đọc dữ liệu blockchain thật
- Tích hợp Ethers.js hoặc Web3.js
- Hỗ trợ nhiều mạng khác nhau
- Thêm biểu đồ số dư và số giao dịch theo thời gian

## Gợi ý trưng bày trên GitHub

Ông chủ nên nhập một địa chỉ ví, chụp giao diện có dữ liệu và thêm ảnh vào README để repo nổi bật hơn.
